# Oasys
Oasys是一个自动化办公系统
## 部署环境
 JDK 1.8+  
 Maven 3.6.0+  
 Docker 20.10.0+  
## 技术栈
 前端：`Vue`  
 后端：`Springcloud`  
 数据库：`Mysql`  
## 关于docker中需要的组件
 Pulsar: 用于实现聊天模块;  
 Nacos: 用于服务发现;  
 MongoDB: 用于储存聊天记录;  
 Minio: 用于储存网盘文件和用户头像;用户默认账号密码minioadmin  
 Redis: 用于缓存对应权限和角色的映射集合;  
 ElasticSearch: 用于搜索，（别忘了安装中文分词插件）
